Progressões Aritmética e Geométrica


PA (Progressão Aritmética)

Uma progressão aritmética é uma sequência formada por termos que se diferenciam um do outro por um valor constante, que recebe o nome de razão, calculado por: r espaço igual a espaço a com 2 espaço subscrito fim do subscrito – espaço a com 1 subscrito

Onde:

    r é a razão da PA;
    a2 é o segundo termo;
    a1 é o primeiro termo.


Sendo assim, os termos de uma progressão aritmética podem ser escritos da seguinte forma:

PA = a1, a2=(a1+r), a3=(a1+2r)... an= [a1 +(n-1)r] Note que em uma PA de n termos a fórmula do termo geral (an) da sequência é: an = a1 + (n – 1) r

Tipos de PA:

Constante:quando a razão for igual a zero e os termos da PA são iguais.

Exemplo: PA = (2, 2, 2, 2, 2, ...), onde r = 0

Crescente: quando a razão for maior que zero e um termo a partir do segundo é maior que o anterior;

Exemplo: PA = (2, 4, 6, 8, 10, ...), onde r = 2

Decrescente: quando a razão for menor que zero e um termo a partir do segundo é menor que o anterior.

Exemplo: PA = (4, 2, 0, - 2, - 4, ...), onde r = - 2

As progressões aritméticas ainda podem ser classificadas em finitas, quando possuem um determinado número de termos, e infinitas, ou seja, com infinitos termos. Considere a PA com o primeiro termo (a₁) igual a 2, a diferença comum (d) igual a 3 e um total de 5 termos (n = 5). Vamos calcular a soma dos termos dessa PA.

A fórmula para a soma dos termos de uma PA é:

Sₙ = (n/2) * (2a₁ + (n-1)d)

Substituindo os valores conhecidos na fórmula, temos:

n = 5
a₁ = 2
d = 3

S₅ = (5/2) * (2*2 + (5-1)*3)
= (5/2) * (4 + 12)
= (5/2) * 16
= 40

Portanto, a soma dos 5 termos dessa PA é igual a 40.


 PA (Progressão Geométrica)

A Progressão Geométrica (PG) é uma sequência de números em que cada termo subsequente é obtido multiplicando-se o termo anterior por uma constante chamada de razão (r). Em outras palavras, cada termo é obtido multiplicando-se o termo anterior por uma mesma quantidade.

A fórmula geral para um termo qualquer (aₙ) de uma PG é dada por:

aₙ = a₁ * r^(n-1)

Onde:

a₁ é o primeiro termo da PG
r é a razão comum da PG
n é o número do termo desejado

Exemplo 1:

Considere a PG com primeiro termo (a₁) igual a 3 e uma razão (r) igual a 2. Vamos calcular o 5º termo dessa PG.

a₅ = 3 * 2^(5-1)
= 3 * 2^4
= 3 * 16
= 48

Portanto, o 5º termo dessa PG é igual a 48.

Exemplo 2:

Agora, vamos calcular a soma dos 4 primeiros termos de uma PG com primeiro termo igual a 2 e razão igual a 3.

A fórmula para a soma dos termos de uma PG finita é dada por:

Sₙ = a₁ * (r^n - 1) / (r - 1)

Substituindo os valores conhecidos na fórmula, temos:

a₁ = 2
r = 3
n = 4

S₄ = 2 * (3^4 - 1) / (3 - 1)
= 2 * (81 - 1) / 2
= 2 * 80 / 2
= 80

Portanto, a soma dos 4 primeiros termos dessa PG é igual a 80.

Matrizes e Determinantes


Uma matriz é uma estrutura organizada de elementos dispostos em linhas e colunas. Cada elemento em uma matriz é chamado de entrada. As matrizes são denotadas por letras maiúsculas, como A, B, C, etc.

Por exemplo, vamos considerar a matriz A de ordem 2x3 (2 linhas e 3 colunas):

A = [ 1 2 3 ]
[ 4 5 6 ]

Nesse exemplo, temos duas linhas (representadas pelos colchetes) e três colunas (representadas pelos espaços entre os elementos). As entradas individuais da matriz são os números dentro da matriz.

Agora, vamos explorar algumas operações com matrizes:

Adição de matrizes:
Para somar duas matrizes A e B de mesma ordem, basta somar os elementos correspondentes. Por exemplo:

A = [ 1 2 3 ]
[ 4 5 6 ]

B = [ 7 8 9 ]
[ 10 11 12 ]

A + B = [ 1+7 2+8 3+9 ]
[ 4+10 5+11 6+12 ]

Multiplicação de matrizes:

Para multiplicar duas matrizes A e B, o número de colunas de A deve ser igual ao número de linhas de B. O produto é obtido multiplicando-se os elementos correspondentes e somando-os. Por exemplo:

A = [ 1 2 ]
[ 3 4 ]

B = [ 5 6 ]
[ 7 8 ]

A * B = [ (15 + 27) (16 + 28) ]
[ (35 + 47) (36 + 48) ]
